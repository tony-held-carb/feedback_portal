Current System Instructions:
Step 1: Try Email as Password

If you haven't changed your password since registration, try using your email address as the password.

Example: If your email is "{{ user.email if user else 'user@example.com' }}", try "{{ user.email if user else 'user@example.com' }}" as the password.
Step 2: Contact Administrator

If the email-as-password doesn't work, contact the portal administrator.

  • Email: portal-admin@carb.ca.gov (example)
  • Subject: "Password Reset Request - {{ user.email if user else 'your-email@example.com' }}"
  • Include: Your email address and reason for reset
Step 3: Administrator Reset

The administrator will reset your password and notify you of the new credentials.

This process typically takes 1-2 business days.
Why This Process?

This is a development/testing system. In production, you'll receive email-based password reset links automatically. For now, we use manual administrator intervention for security and simplicity.

Security Notes:
  • Never share your password with anyone
  • Use a strong, unique password
  • Consider changing your password after first login
  • Contact support immediately if you suspect unauthorized access
