About This Application

This is a demonstration Flask application that showcases the ARB Auth package with multiple roles support.

Key Features:
  • Multiple Roles: Users can have multiple roles (e.g., editor + qaqc)
  • Role-Based Access: Different pages require different role combinations
  • Flexible Decorators: Support for single roles, any roles, or all roles
  • User Management: Admin interface for managing users and roles
  • Template Integration: Role checking in Jinja2 templates
Example Role Combinations:
  • editor - Can access editor tools
  • qaqc - Can access QA/QC tools
  • editor, qaqc - Can access both editor AND advanced editing
  • reviewer, manager - Can access review panel
  • admin - Full administrative access
Getting Started

To explore the application:

  1. Register a new account or log in
  2. Visit your dashboard to see available features
  3. Try accessing different sections based on your roles
  4. If you're an admin, manage other users' roles
